Key Features

person

Instant order verdict

Open an order in Odoo or Shopify and the panel appears with a global verdict — match or mismatch — before you finish reading the page.

dashboard

Field-by-field breakdown

Totals, taxes, quantities, line items and addresses compared side by side, with every discrepancy called out and explained.

mail

Verdict based on what's invoiced

Compares Shopify's current order against Odoo's net invoiced amount (invoices minus refunds) — not the frozen sale order — so partial invoices and refunds don't trigger false alarms.

sync

Refunds & returns in context

Refunds, returns and order edits are shown as context, never counted as discrepancies. You see why the numbers differ, not just that they do.

business

Zero configuration

Auto-detects your Odoo instance and database from your active session. No instances to register, no API keys, no options page.

bolt

Read-only & safe

The extension never creates, edits or deletes anything in Odoo or Shopify. It only reads — safe to run on production from day one.
